## 🎯 About

This repository contains **five core Python projects** built from scratch with **zero dependencies**. Each project demonstrates practical application of Python concepts including Object-Oriented Programming (OOP), data validation, and real-world problem-solving.

## 📁 Projects

### 💰 [BudgetApp.py](./BudgetApp.py)
**Budget Tracker Application**
- 📊 Track your expenses and income
- 💳 Manage multiple budget categories
- 📈 View financial summaries
- ⚙️ OOP-based budget management system

### 📧 [EmailSimulator.py](./EmailSimulator.py)
**Email System Simulator**
- ✉️ Simulate sending and receiving emails
- 📬 Manage inbox and sent folders
- 👥 User authentication system
- 🔐 Email validation and filtering

### 📚 [MediaCatalogue.py](./MediaCatalogue.py)
**Media Library Manager**
- 🎬 Organize movies, shows, and videos
- 🏷️ Categorize and tag media items
- 🔍 Search and filter functionality
- ⭐ Rating and review system

### ✅ [MedicalDataValidator.py](./MedicalDataValidator.py)
**Medical Data Validation System**
- 🏥 Validate patient information
- 📋 Data integrity checking
- 🔒 Privacy-compliant validation
- 📊 Report generation capabilities

### 🔷 [PolgonAreaCalculator.py](./PolgonAreaCalculator.py)
**Geometry Calculator**
- 📐 Calculate polygon areas
- 🔢 Support for various shapes
- 📏 Mathematical computation engine
- 🎨 Shape manipulation and analysis

### 🔗 [Link_List.py](./Link_List.py)
**Linked List Data Structure**
- ⛓️ Implement linked list operations
- 🔄 Traversal and manipulation
- 📦 Node-based data organization
- 🧠 Fundamental data structure learning

### 🏔️ [hanoi.py](./hanoi.py)
**Tower of Hanoi Solver**
- 🎯 Classic algorithm implementation
- 🎲 Recursive problem solving
- 📜 Move sequence generation
- 🧩 Algorithmic thinking

### 🗺️ [shortest_path.py](./shortest_path.py)
**Shortest Path Algorithm**
- 🛣️ Find optimal routes
- 📍 Graph traversal
- ⚡ Path optimization
- 🧭 Navigation algorithms

## 🚀 Getting Started

### Prerequisites
- Python 3.7 or higher
- No external dependencies needed!

### Installation

```bash
# Clone the repository
git clone https://github.com/priyankardas2007/Learning_Python.git

# Navigate to the project directory
cd Learning_Python
```

### Running a Project

```bash
# Run any Python file
python BudgetApp.py
python EmailSimulator.py
python MediaCatalogue.py
# ... and so on
```

- **Inheritance** - Code reusability and hierarchy
- **Encapsulation** - Data protection and organization
- **Polymorphism** - Flexible method implementations

### ✔️ Data Validation
